### Dataset
The dataset contains global AI and machine learning salary data collected from professionals between 2020 and 2022. It includes information such as job titles, employee residence, salary converted to USD, and work year, covering 64 countries and over 1,300 records. The data was sourced from ai-jobs.net and reflects real-world compensation trends in the AI/ML job market. It is used in this project to analyze salary patterns across countries and identify cost-effective regions for global hiring strategies.
